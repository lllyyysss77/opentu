欢迎使用 Opentu（开图）AI 应用平台

Opentu（开图） 是一个以浏览器为入口的 AI 应用平台，围绕画布工作区把模型、知识、素材与工具协作串成一个完整工作台。 平台以画布工作区为运行底座，用户在同一个画布中发起 AI 绘图、AI 视频生成、流程图、思维导图、知识卡整理与任务队列管理，并通过多模型路由完成图像、视频、文本等交付。

这让创作者不用在多个应用间切换，就能在一个开放工作区内完成从灵感到交付的全链路。

注意：重要提示： 您的所有数据（项目、素材、提示词历史）都保存在当前浏览器的本地存储中，不会上传到服务器。这意味着：
  • ✅ 数据隐私安全，完全由您掌控
  • ⚠️ 换浏览器或清除浏览器数据会丢失内容
  • 💡 需要迁移数据？使用 导出与导入 页介绍的 菜单 → 备份恢复 功能导出后在新浏览器导入

Opentu 平台概览

  • AI 应用 - 在底部 AI 输入栏发起图片、视频、灵感板的生成请求，或通过 Agent/Skill 调用多模型能力
  • 知识与素材 - 知识库、灵感板、素材库都与画布互联，生成内容可一键插入工作区并被任务队列追踪
  • 工具与工作流 - 命令面板、工具箱与工作流系统贯穿整个平台，支持多步骤执行与任务重试
  • 开放与扩展 - 插件架构、API 配置与同步能力让 Opentu 适合作为高频 AI 工作空间

产品展示

智能拆分图片

输入描述，AI 生成可拆分的图片，一键拆成九宫格，发朋友圈超方便。想看完整操作，可继续阅读 九宫格拆图指南。

流程图

用自然语言描述流程，AI 自动生成专业流程图；如果你已经有 Mermaid 代码，也可以直接导入，详见 AI 生成流程图与 Mermaid 转流程图。

思维导图

输入主题，AI 帮你拓展思路，快速生成完整的思维导图；若你更习惯写提纲，还可以使用 Markdown 转思维导图。

我能用它做什么？

平台能力 🔥

  • AI 生成图片 - 支持文生图、参考图生成、批量生图，可切换 nano-banana、nano-banana-2 等模型
  • AI 生成视频 - 支持文生视频、图生视频、首尾帧控制，可使用 Veo3、Veo3.1、Sora-2 等模型
  • 模型自由切换 - 在生成对话框中即时切换不同的 AI 模型，按应用场景选择图片模型或视频模型
  • 任务队列管理 - 异步任务处理，支持批量生成、任务重试、历史记录、媒体缓存

画布工作区与可视化

  • 思维导图 - 快速整理思路，支持 AI 生成脑图与 Markdown 转思维导图
  • 流程图 - 绘制业务流程图，支持 Mermaid 转流程图
  • 自由画笔 - 手绘涂鸦，多种画笔可选
  • 形状工具 - 矩形、圆形、箭头等基础形状
  • 图片插入 - 支持插入和编辑图片元素

编辑与交互

  • 丰富的编辑功能 - 撤销、重做、复制、粘贴、多选等
  • 无限画布 - 自由缩放、滚动、移动
  • 自动保存 - 本地浏览器自动保存，数据不丢失
  • 多格式导出 - 支持 PNG、JSON(.drawnix) 等格式导出，也支持项目备份恢复

体验与生态

  • 完全免费 + 开源 - MIT 许可证，可商用
  • 主题模式 - 支持亮色/暗色主题切换
  • API 配置与移动端适配 - 支持 API Key 配置、主题切换与移动设备使用

立即开始

访问 opentu.ai 开始使用，无需注册账号。

体验新功能？试试 pr.opentu.ai

快速入门

1. 生成图片

在底部输入框输入描述（如"一只可爱的猫咪"），点击发送。可以先参考 AI 生成图片指南 选择模型和尺寸。

2. 绘制思维导图

点击工具栏的思维导图图标，或按 M 键，开始创建。若已有提纲内容，可直接用 Markdown 转思维导图。

3. 自由绘画

按 P 键切换到画笔，在画布上自由绘制。

界面介绍

  • 顶部工具栏 - 各种绘图工具和编辑功能
  • 底部输入框 - AI 创作入口，输入描述生成图片/视频
  • 右下角导航 - 缩放、重置视图等操作
  • 无限画布 - 可自由拖动、缩放的创作空间

数据存储说明

您的数据存储在浏览器本地，包括：

数据类型存储位置说明
项目文件IndexedDB画布内容和结构
素材库Cache Storage上传和生成的图片视频
提示词历史IndexedDB使用过的 AI 提示词
设置LocalStorageAPI 配置等
数据迁移方法：

1. 在原浏览器打开 导出与导入 中介绍的 菜单 → 备份恢复 2. 勾选需要迁移的内容，点击 导出 3. 在新浏览器打开 Opentu，导入 刚才的 ZIP 文件
